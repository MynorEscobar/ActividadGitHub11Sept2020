Concepto de GIT
Es un Software que cotrola versiones de escritura creado por Linus Troval, GIT da la oportunidad de trabajar en equipo de una manera eficiente y rápida.
Concepto GITHUB
Se puede decir que es entonces la plataforma donde se alojan los proyectos en el cual se utiliza el contro de versiones GIT, es entonces una compañia que ofrece el alojamiento sin fines lucrativos
